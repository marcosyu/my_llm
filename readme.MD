## MY AI Stack ##
----------------------------------------
This project aims to provide an efficient and effective toolkit that supports both research & general inquiries with or without the internet, as well specific code generation in bulk using various LLMs such as Mistral default, Deepseek-Coder (code specfic),OLLaMA3.

### AI Companion ###
AI companion provides a systematic way to help users understand and navigate through complex topics with or without the internet connectivity enabled on their devices by:
1) Gathering data from various sources in real-time (without needing an Internet Connection). This will enable researchers and developers alike, more quickly access relevant information.
2) Helping general inquiries to generate code snippets as per user preference without the need for manual input of codes themselves - enhancing efficiency by reducing coding errors overheads or time spent debugging later on in development process flow due a broader range with multiple AI tools and platforms at their disposal, enabling faster problem-solving.
  LLMs: Mistral (default), Deepseek Coder(code specific) & LLaMA3 are commonly used for different types of questions related to computer science or programming as they provide natural language understanding capabilities that can be highly beneficial in both research and general tasks with the aid of AI assistants like this.

### Code Generation from Bulk ###
For those who require a bulk creation process, we've integrated an Agent Orchestration Framework - LangChain/LangGraph into our stack to enable code generation in multiple languages at once:
- A multi language (ML) agent orchestrator enables managing complex tasks across different programming environments. It ensures consistency and seamless functioning of the generated codes regardless of any changes made on one environment, aiding productivity enhancement as well ensuring efficient error handling due various AI tools & platforms available for multiple languages at once in our case 'LangChain / LangGraph'.
